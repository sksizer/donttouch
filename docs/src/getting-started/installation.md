# Installation

## From Source (Cargo)

```bash
cargo install donttouch
```

## From GitHub

```bash
git clone https://github.com/sksizer/donttouch
cd donttouch
cargo install --path .
```

## npm (coming soon)

```bash
npx donttouch init
```

The npm package will download the appropriate binary for your platform.

## Verify

```bash
donttouch --version
```
